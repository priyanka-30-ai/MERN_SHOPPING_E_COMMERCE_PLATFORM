A full-stack **E-Commerce web application** built using the **MERN stack**
(MongoDB, Express, React, Node.js).

The platform allows users to browse products, add items to cart, place orders,
and manage their accounts. An admin panel is included for managing products
and orders.

---

## Features

### User Features
- User authentication (Login & Register)
- Browse products with detailed views
- Add products to cart
- Place orders
- View order history

### Admin Features
- Admin authentication
- Add, update, and delete products
- Manage customer orders
- Upload product images

---

## Tech Stack

### Frontend
- React
- Tailwind CSS
- Vite

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Other Tools
- Cloudinary (Image uploads)
- JWT Authentication

## How to Run the Project Locally

### 1. Clone the repository
```bash
git clone https://github.com/priyanka-30-ai/MERN_SHOPPING_E_COMMERCE_PLATFORM.git
```

### 2. Install backend dependencies
```bash
cd backend
npm install
```

### 3. Install frontend dependencies
```bash
cd ../frontend
npm install
```

### 4. Install admin panel dependencies
```bash
cd ../admin
npm install
```

### 5. Start the backend server
```bash
cd ../backend
npm start
```

### 6. Start frontend and admin panel
```bash
cd ../frontend
npm start
```

```bash
cd ../admin
npm start
```

### 7. Open in browser
```
http://localhost:3000
```
